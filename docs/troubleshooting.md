# 故障排除

## 安装/部署问题

| 问题 | 原因 | 解决 |
|:-----|:-----|:-----|
| 技能没出现 | 未安装或未重启 | 运行 `.\install.ps1`，重启 Codex |
| 部分技能缺失 | 子目录未正确复制 | 重新运行 `.\install.ps1` |
| solo-law-firm 没装 | 嵌套目录处理异常 | 检查 `skills/solo-law-firm/` 是否存在 |

## 运行问题

| 问题 | 原因 | 解决 |
|:-----|:-----|:-----|
| 引用标注 [需验证] | MCP 未配置 | 运行 `.\update.ps1` 查看 MCP 状态 |
| 技能说英文 | CLAUDE.md 未中文化 | 检查 `skills/*/CLAUDE.md` 是否为中文 |
| 法条引用是美国的 | 技能调用了原版 | 检查 `patches/sub-skills/` 快照是否最新 |

## 上游检测问题

| 问题 | 原因 | 解决 |
|:-----|:-----|:-----|
| diff-tool 报 git 错误 | TEMP 目录冲突 | 删除 `$env:TEMP\zhou-check` 重试 |
| 子技能全是 [+] | 首次运行，无快照 | 执行 `-Update` 创建快照 |
| 子技能全是 [Δ] | 快照来自本地而非上游 | 重新运行 `-Update`（从上游创建） |
| 上游监控无通知 | 本周无新 commit | 等待下周一或手动触发 Actions |

## 性能问题

| 问题 | 原因 | 解决 |
|:-----|:-----|:-----|
| diff-tool 运行慢 | 首次 git clone | 之后有 TEMP 缓存，会快很多 |
| install/update 慢 | 克隆 MCP Hub | 之后有本地缓存，会快很多 |
