# 连接器指南（MCP）

法律技能在连接了权威数据源后效果最佳。本仓库预配置了主流中国法律数据库的 MCP 连接器支持。

## 预配置连接器

| 连接器 | 适用领域 | 用途 |
|--------|---------|------|
| **yuandian（元典）MCP** | 全部 12 个领域 | 案例检索、法规检索 |
| **北大法宝 MCP** | litigation-legal, ip-legal, law-student | 法规库、案例库、法学期刊 |

## 为什么需要连接器

没有连接法律检索工具时，技能引用的法规和案例来自模型训练数据。连接器确保引用的是当前有效的法律依据。未连接时，技能输出中的引用会被标注为 **[需验证]**。

## 配置方法

连接器配置位于各领域技能目录的 `.mcp.json` 文件中。第一次使用时系统会提示你授权连接。

### 手动配置

1. 打开技能目录下的 `.mcp.json`
2. 填入你的 API Key 或认证信息
3. 重启 Codex Desktop

## 推荐连接器

| 服务商 | 覆盖内容 | 适合领域 |
|--------|---------|---------|
| **元典（yuandian）** | 案例、法规 | 全部领域 |
| **北大法宝** | 法规、案例、期刊 | 诉讼、知产、法考 |
| **威科先行** | 法规、裁判文书、实务指南 | 商事、劳动、合规 |

## 无连接器时的行为

即使不配置任何连接器，技能仍可正常使用。模型会基于训练数据中的法律知识提供分析，但：

- 法规引用将标注 **[需验证现行有效性]**
- 案例引用可能不包含最新判例
- 建议在实际使用前通过人工检索核实
