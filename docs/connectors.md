# 连接器指南（MCP）

法律技能在连接了权威数据源后效果最佳。

## 预配置连接器

- **yuandian（元典）MCP** — 全部 12 个领域，案例检索和法规检索
- **北大法宝 MCP** — litigation-legal, ip-legal, law-student 领域

## 为什么需要连接器

没有连接法律检索工具时，技能引用的法规和案例来自模型训练数据。
连接器确保引用的是当前有效的法律依据。未连接时引用会被标注为[需验证]。

## 配置方法

连接器配置位于各领域技能目录的 .mcp.json 文件中。
第一次使用时系统会提示你授权连接。

## 推荐连接器

1. yuandian（元典）MCP — 案例检索和法规检索
2. 北大法宝 MCP — 法规库、案例库、法学期刊
3. 威科先行 MCP — 法律法规、裁判文书、实务指南
