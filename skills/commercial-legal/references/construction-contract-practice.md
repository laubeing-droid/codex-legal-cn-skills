# 建设工程施工合同法律实务知识框架

## 一、建设工程合同概述

### 定义（《中华人民共和国民法典》第788条）
建设工程合同是承包人进行工程建设，发包人支付价款的合同，包括工程勘察、设计、施工合同。

### 特征
- 持续性履行合同 — 事实复杂且在诉讼中易变化
- 国家管控和干预程度深
- 法律关系纠集：建工合同、租赁合同、劳务合同等交织
- 脱法行为和隐藏行为多

### 主要法律依据
| 法律文件 | 施行时间 |
|:---------|:---------|
| 《民法典》 | 2021.1.1 |
| 《建筑法》 | 2019.4.23修订 |
| 《招标投标法》 | 2017.12.28修订 |
| 最高法《建工解释一》 | 2005.1.1 |
| 最高法《建工解释二》 | 2019.2.1 |
| 《建设工程价款结算暂行办法》 | 2004.10.20 |

## 二、合同订立

### 成立时间
- 一般：中标通知书送达时合同成立，事后签订合同书是合同的表现形式
- 特殊（挂靠）：当事人合意形成时间为合同成立时间

### 挂靠（前挂）
三层合同关系：
1. 发包人与挂靠人 — 事实上的建设工程合同（真实意思表示但违法）
2. 发包人与被挂靠人 — 形式上的合同（表面合法，虚伪意思表示）
3. 挂靠人与被挂靠人 — 挂靠合同（真实意思表示但违法）

> 诉讼建议：三方当事人均需参加诉讼，以便查清事实、分清责任。

### 非典型挂靠（后挂）
- 发包人不知道挂靠事实 → 按转包处理
- 发包人知道挂靠 → 按挂靠处理

### 实际施工人权利
2004年《建工解释一》第26条：实际施工人可以突破合同相对性，直接向发包人主张权利（发包人在欠付工程款范围内承担责任）。

### 特殊名称合同
"内部承包合同""合作合同"等——关键是看合同内容而非名称判断合同性质。合作/联营需共担风险、共同参与且具有资质，否则可能构成转包。

## 三、合同效力

### 无效情形
- 承包人未取得建筑业企业资质或超越资质等级
- 没有资质的实际施工人借用有资质企业名义（挂靠）
- 建设工程必须招标而未招标或中标无效
- 转包
- 违法分包

### 无效后果
- 合同无效但工程验收合格 → 可参照合同约定支付工程价款
- 合同无效且工程验收不合格 → 修复后合格可主张工程款（修复费用由承包人承担）；修复后仍不合格 → 无权主张工程款

## 四、工程价款

### 结算依据
| 情形 | 结算依据 |
|:-----|:---------|
| 合同有效 | 合同约定 |
| 合同无效但验收合格 | 参照合同约定 |
| 多份合同（黑白合同） | 中标备案合同 |
| 无书面合同 | 实际履行 + 市场价格/计价标准 |

### 价款优先受偿权（《中华人民共和国民法典》第807条）
- 发包人未按约定支付价款的，承包人可以催告发包人在合理期限内支付
- 发包人逾期不支付的，承包人可与发包人协议将该工程折价，或请求法院拍卖
- 建设工程的价款就该工程折价或拍卖的价款优先受偿
- **行使期限**：自发包人应当给付建设工程价款之日起18个月内行使（原6个月→改为18个月）

### 实际施工人权利
- 实际施工人可突破合同相对性起诉发包人
- 发包人在欠付转包人/违法分包人工程款范围内承担连带责任

## 五、工期

### 开工日期认定
- 开工通知载明的日期
- 开工通知发出后不具备施工条件 → 条件具备日期
- 承包人实际进场施工日期

### 竣工日期认定
- 竣工验收合格 → 竣工验收合格之日
- 发包人拖延验收 → 承包人提交验收报告之日
- 未经竣工验收擅自使用 → 转移占有之日

### 工期延误
- 发包人原因 → 顺延 + 赔偿
- 承包人原因 → 违约金/赔偿
- 不可抗力 → 顺延

## AI 审查建工合同的核查清单

| # | 核查项 | 风险 |
|:--|:-----|:----|
| 1 | 承包人资质是否与工程等级匹配？ | 🔴 无资质/超越资质→合同无效 |
| 2 | 是否属于必须招标项目？是否已招标？ | 🔴 应招未招→合同无效 |
| 3 | 是否存在挂靠/转包/违法分包？ | 🔴 合同无效风险 |
| 4 | 工程价款支付节点是否明确？ | 🟡 模糊条款→结算争议 |
| 5 | 是否约定价款优先受偿权放弃？ | 🔴 损害建筑工人利益→无效 |
| 6 | 开工/竣工日期认定标准是否明确？ | 🟡 工期争议 |
| 7 | 质保金比例是否合法？（≤3%工程价款总额） | 🟢 合规检查 |
| 8 | 违约金是否过高？（>LPR 4倍可申请调减） | 🟡 违约金调减风险 |
