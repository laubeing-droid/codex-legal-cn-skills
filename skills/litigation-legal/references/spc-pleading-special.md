# 特殊程序要素式文书模板集

> 来源：最高人民法院示范文本 — 海事/环境公益/刑事自诉/国家赔偿/执行申请

---

## 一、海事案件（4 类）

| 案由 | 特有要素 |
|:-----|:---------|
| 海上人身损害 | 事故时间/地点/船名/伤亡情况/医疗费/误工费/伤残赔偿金 |
| 海上货运代理 | 委托事项/费用标准/欠付金额/货物损失 |
| 船员劳务合同 | 上船时间/职务/工资标准/欠薪金额/遣返费用 |
| 船舶碰撞 | 碰撞时间/地点/船名/责任认定/船舶损失/货物损失 |

---

## 二、环境公益诉讼（3 类）

### 环境污染/生态破坏民事公益诉讼

| # | 要素 |
|:--|:-----|
| 1 | 污染/破坏行为（时间/地点/方式/污染物种类和数量） |
| 2 | 生态环境损害后果 |
| 3 | 因果关系 |
| 4 | 是否请求修复生态环境 |
| 5 | 是否请求赔偿损失（生态环境服务功能损失/调查评估费用/律师费/诉讼费） |
| 6 | 是否申请行为保全（禁止令） |

---

## 三、刑事自诉（4 类）

### 侮辱/诽谤/重婚/拒不执行判决裁定

| # | 共通要素 |
|:--|:-----|
| 1 | 被告人身份信息 |
| 2 | 犯罪时间/地点/手段 |
| 3 | 自诉人因被告人犯罪行为所受损害 |
| 4 | 证据清单（证人/书证/物证/视听资料/电子数据） |
| 5 | 是否提起附带民事诉讼（赔偿请求+计算方式） |

### 各案由特有

| 案由 | 特有要素 |
|:-----|:---------|
| 侮辱 | 侮辱方式（口头/书面/网络）/是否公然/是否情节严重 |
| 诽谤 | 诽谤内容/传播范围/是否捏造事实 |
| 重婚 | 被告人已婚事实/重婚时间/重婚对象 |
| 拒执 | 生效判决案号/判决内容/被执行人有能力履行而拒不履行的事实 |

---

## 四、国家赔偿（4 类）

### 通用要素

| # | 要素 |
|:--|:-----|
| 1 | 赔偿义务机关名称 |
| 2 | 侵权事实（时间/行为/结果） |
| 3 | 申请赔偿的法律依据和理由 |
| 4 | 有无自赔决定（决定书文号/时间/结果） |
| 5 | 有无复议决定 |

### 各类型赔偿金计算

| 类型 | 赔偿项目 |
|:-----|:---------|
| 刑事改判无罪 | 人身自由赔偿金（天数×国家上年度日平均工资）+ 精神损害赔偿（消除影响/恢复名誉/赔礼道歉/抚慰金） |
| 违法刑事拘留 | 同上 |
| 错误执行 | 财产权赔偿（返还原物/恢复原状/赔偿损失）+ 停产停业损失（留守工资/税款社保/水电仓储租金/设备折旧） |
| 怠于履职致伤致死 | 侵犯生命健康权赔偿金 + 精神损害赔偿 |

## 五、执行申请（9 类）

| 文书 | 核心要素 |
|:-----|:---------|
| 强制执行申请书 | 执行依据（判决/裁定/调解书/公证债权文书案号）+ 申请执行标的 + 被执行人财产线索 |
| 执行异议申请书 | 异议人信息 + 执行案号 + 异议事由 + 法律依据 |
| 执行复议申请书 | 原异议裁定案号 + 复议事由 + 法律依据 |
| 执行监督申请书 | 执行法院 + 执行案号 + 违法/不当执行行为 + 请求监督事项 |
| 执行担保申请书 | 担保人信息 + 担保财产 + 担保范围 + 担保期限 |
| 参与分配申请书 | 执行案号 + 债权依据 + 债权金额 + 被执行人财产情况 |
| 不予执行仲裁裁决申请书 | 仲裁裁决案号 + 不予执行事由（仲裁程序违法/超裁/伪造证据/隐瞒证据/违背社会公共利益） |
| 确认优先购买权申请书 | 执行标的 + 优先购买权依据（共有/承租人/股东） |
| 暂时解除限高申请书 | 限高令案号 + 申请事由（本人/近亲属重大疾病就医/近亲属丧葬/执行公务/外事活动/重要考试） |

## AI 输出约束
- ✅ 国家赔偿按《中华人民共和国国家赔偿法》第33条（日赔偿金=国家上年度日平均工资）
- ✅ 执行申请文书格式按照最高法执行文书样式
- ✅ 刑事自诉需审查是否属于告诉才处理的案件（侮辱/诽谤/暴力干涉婚姻自由/虐待/侵占）
- ⛔ 禁止引用美国法下的 Discovery/Injunction/Habeas Corpus 等概念
