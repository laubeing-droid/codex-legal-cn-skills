# 地域裁判口径查询 — 按需对比工具

## 触发条件
用户询问"XX地区怎么看""各地对XX怎么判""XX争议南北差异"等地域对比问题时触发。

## 查询流程

### 1. 确定争议点
从用户问题中提取：案由 + 争议焦点 + 法条

### 2. 五地并行检索
通过元典智库 MCP，对每个地域执行类案检索：

| 地域 | 检索关键词模板 |
|:-----|:-----|
| 北京 | `北京 高院 {争议点} 指导意见 OR 典型案例` |
| 上海 | `上海 高院 {争议点} 裁判纪要 OR 参考案例` |
| 广东 | `广东 高院 深圳 {争议点} 审判指引` |
| 浙江 | `浙江 高院 {争议点} 问题解答 OR 指导意见` |
| 最高法 | `{争议点} 指导案例 OR 公报案例 OR 司法解释` |

### 3. 结果对比
检索结果按以下维度对比：

| 对比维度 | 说明 |
|:-----|:-----|
| 法律依据 | 各地引用的法条/司法解释是否一致 |
| 裁判规则 | 各地是否有明文指导意见？内容差异？ |
| 实务倾向 | 类案检索中支持/驳回比例 |
| 定量差异 | 赔偿额/违约金/刑期的各地区间 |

### 4. 输出格式

```markdown
## {争议点} — 五地裁判口径对比

> 检索时间：[]

### 最高法层面
- 法律依据：[]
- 指导案例/司法解释：[]

### 各地口径
| 地域 | 裁判规则 | 实务倾向 | 依据来源 |
|:-----|:---------|:---------|:-----|
| 北京 | | | |
| 上海 | | | |
| 广东 | | | |
| 浙江 | | | |

### 实务建议
- 对原告：[地域选择/诉讼策略建议]
- 对被告：[抗辩重点/有利地域提示]
- ⚠️ 以上基于类案检索统计，非确定性结论

### 数据来源
- 元典智库类案检索结果
- 各高院指导意见/裁判纪要（如有）
```

## 检索限制
- 每次查询最多返回 5 个案例/地域
- 标注检索时间（裁判口径可能变化）
- 无明确依据时标注"未检索到相关文件，以下为类案倾向判断"
